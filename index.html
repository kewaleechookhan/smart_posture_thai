ลำดับหลักตั้งแต่ผู้เข้าร่วมทำแบบประเมินจนถึงการติดตามผลและสรุปรายงาน

1ผู้เข้าร่วมทำแบบประเมินผ่าน Web Appสแกน QR Code หรือเปิดลิงก์เฉพาะของหน่วยงาน
2บันทึกข้อมูลเข้า Google Sheetรองรับ Apps Script endpoint หรือจัดเก็บในเครื่องเพื่อ demo
3คำนวณคะแนนอัตโนมัติรวมคะแนน risk, pain, activity และ ergonomics โดยแยก satisfaction เป็นข้อมูลประกอบ
4Dashboard รายบุคคล/ภาพรวมแสดงระดับความเสี่ยง ค่าเฉลี่ย และ KPI
5ติดตามผลและสรุปรายงานใช้ PreTest, PostTest และ FollowUp 2 สัปดาห์
