# SMART POSTURE THAI - คู่มือนำไปใช้งานจริง

## ไฟล์ในชุดนี้
- `index.html` = หน้าเว็บสำหรับบุคลากร/ผู้รับการประเมิน ใช้ไฟล์นี้สำหรับอัปโหลดขึ้นเว็บจริง
- `admin-local.html` = ไฟล์เต็มสำหรับผู้จัดทำเปิดตรวจในเครื่อง ไม่ควรใช้เป็นลิงก์สาธารณะ
- `smart-posture-thai-apps-script.gs` = โค้ด Google Apps Script สำหรับรับข้อมูลเข้า Google Sheet

## ขั้นตอนใช้งานจริงแบบแนะนำ
1. เปิด Google Sheet ที่เตรียมไว้ และตรวจว่า Apps Script Web App URL ยังใช้งานได้
2. นำไฟล์ `index.html` ไปอัปโหลดบนเว็บโฮสติ้ง เช่น Netlify Drop, Vercel, GitHub Pages หรือโฮสติ้งของหน่วยงาน
3. หลังอัปโหลดแล้ว ให้เปิด URL จริง เช่น `https://.../index.html`
4. QR Code และลิงก์ในหน้าเว็บจะเปลี่ยนตาม URL จริงโดยอัตโนมัติ
5. ทดลองกรอกข้อมูล 1 รายการ กด `ส่งคำตอบให้ผู้ประเมิน`
6. เปิด Google Sheet ตรวจว่าข้อมูลเข้าแท็บ `Participants` และ `PreTest/PostTest/FollowUp`
7. หากทดสอบผ่าน จึงส่งลิงก์หรือ QR Code ให้บุคลากรใช้งานจริง

## ช่องทางอัปโหลดที่ง่ายที่สุด
Netlify Drop:
1. เข้า https://app.netlify.com/drop
2. ลากไฟล์ `index.html` หรือโฟลเดอร์ชุดนี้ไปวาง
3. ระบบจะสร้าง URL ให้ทันที
4. ใช้ URL นั้นเป็นลิงก์จริงสำหรับส่งให้บุคลากร

## ข้อควรระวัง
- อย่าใช้ลิงก์ `file:///...` ส่งให้บุคลากร เพราะเปิดได้เฉพาะเครื่องของผู้จัดทำ
- อย่าเผยแพร่ `admin-local.html` เป็นลิงก์สาธารณะ เพราะเป็นหน้าเต็มสำหรับผู้จัดทำ
- ข้อมูลสุขภาพควรเก็บใน Google Sheet ที่จำกัดสิทธิ์เฉพาะผู้รับผิดชอบโครงการ
- หากเปลี่ยน Google Apps Script URL ต้องแก้ในไฟล์ `index.html` ก่อนอัปโหลดใหม่
